About BohConf

Back to top

BohConf is the official RailsConf 2010 unconference. At BohConf, we're going to get our hands dirty writing code and sharing ideas in an open and free-form environment. This hacking-centric event will include community code drives featuring well known OSS authors, a code retreat, a programming competition and barcamp-style discussions. It is free and will run alongside RailsConf in the convention center. Everyone is welcome at BohConf. Registration is free and can be completed at https://en.oreilly.com/rails2010/public/register

You can track updates and developments by following us on Twitter and by contributing to the BohConf wiki. Your input will shape this event.

What is Boh?

Back to top

The name BohConf is an homage to the Baltimore institution National Bohemian (or Natty Boh). Every town has their local light beer of choice, and ours is Natty Boh. Make sure to grab one (or several) when you're in town.

Sessions

Back to top

Community Code Drive

The Community Code Drive is a place to meet and work with the authors of many popular open source projects. Come to help hack on code, find and fix bugs, or just watch a seasoned developer at work. Check out the schedule to see which OSS authors are participating (and feel free to add yourself)!

Code Retreat

Code. Rinse. Repeat. As professional developers we're often coding for profit and rarely coding for practice. In a Code Retreat we'll sharpen our coding skills with a set of exercises done in pairs, and we'll start fresh every time.

Programming Contest

Come flex your coding muscles with a data visualization challenge! We'll unveil a dataset and give participants 24 hours to do whatever they want with it. It doesn't have to be rails (or even a website) but it does have to be cool! Prizes will include a $100 gift card, an assortment of O'Reilly books, and BohConf T-Shirts.

Ruby and Art

Do you like ruby? Do you like art? Do you like taking two things that you like, and then sticking them together to make a bigger thing you like even more? We'll be experimenting with visual art, music, games, and of course, Ruby!

Barcamp

A Barcamp is a "user generated conference", so there's structure without an agenda. Bring your own questions and your own answers, and we'll hold our own BohCamp.
